# WPF Loaders & To-Do List

A WPF application built with Caliburn.Micro and MVVM pattern.

## Features

- **Loaders** – Three threads running simultaneously, each with random duration (10–50s), real-time progress tracking and cancel support
- **To-Do List** – Add tasks with priority (1–3), auto-sorted and color-coded list

## Tech Stack

- WPF / C#
- Caliburn.Micro
- MVVM
- MSTest

## Getting Started

1. Open `Assignment.sln` in Visual Studio
2. Build and run the `Assignment` project

## Running Tests

Open Test Explorer in Visual Studio and run all tests in `Assignment.Tests`.
